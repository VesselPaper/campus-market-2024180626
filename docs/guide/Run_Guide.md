# 项目运行说明

## 环境要求

- Node.js >= 18
- npm

## 启动步骤

### 方式一：手动启动（推荐）

**1. 启动 Mock API 服务**

```bash
cd campus-market
npx json-server --watch db.json --port 3000
```

验证: 浏览器访问 http://localhost:3000/items 能看到 JSON 数据。

**2. 启动前端应用**

```bash
# 新开一个终端
cd campus-market
npx vite --port 5173
```

验证: 浏览器访问 http://localhost:5173 能看到校园轻集市页面。

### 方式二：一键启动

双击运行 `scripts/start.bat`，会自动启动两个服务。

## 操作指南

1. **创建身份** — 首次进入填写昵称、学院、校区、角色
2. **浏览首页** — 查看欢迎信息、快捷入口、最新发布
3. **集市列表** — 筛选、搜索、排序浏览信息
4. **查看详情** — 点击卡片查看完整信息，可收藏/联系发布者
5. **发布信息** — 选择类型填写表单后发布
6. **消息中心** — 查看会话、发送消息
7. **个人中心** — 管理发布和收藏，更新状态
8. **趋势看板** — 查看数据可视化图表

## 常见问题

**Q: 页面空白或数据不显示？**
A: 请确认 JSON Server 已在端口 3000 启动。

**Q: 创建身份失败？**
A: 同样检查 JSON Server 是否运行。

**Q: 端口被占用？**
A: 修改启动命令中的端口号，如 `--port 5174`。

## 端口说明

| 服务 | 默认端口 | 说明 |
|------|----------|------|
| Vite 前端 | 5173 | 浏览器访问 |
| JSON Server | 3000 | Mock API |
